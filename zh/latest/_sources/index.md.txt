# TuFT

<p class="hero-subtitle"><strong>TuFT (Tenant-unified Fine-Tuning)</strong> 是一个多租户平台，支持多用户通过统一 API 在共享基础设施上微调大语言模型。可通过 Tinker SDK 或兼容客户端接入使用。</p>
